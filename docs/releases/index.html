Version 0.1 #

The first release of BulmaWP is here! Lot's of work has been put into this free theme but there is still lots more to do. As you can see, there is also an official website for BulmaWP so that in the future it's easy to download releases and view the docs.

Here's a short summary of everything that has been added to the theme.

  • Breadcrumbs
  • Font Awesome 5
  • Footer
  • Navbar
  • Pagination
  • Search bar
  • Settings page to customise the theme
  • Sidebar
  • Social media buttons
