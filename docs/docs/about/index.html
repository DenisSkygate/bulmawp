About BulmaWP

BulmaWP is a project that I am constantly working on to improve and to further my knowledge within WordPress development. As a junior web developer, I believed that this would give me a deep insight into developing WordPress themes that are complaint with the WordPress standards and recommendations.

This theme doesn't include that much customisation simply because I wanted it to be more of a foundation to which developers could combine the functionality of WordPress with the Bulma CSS framework without building themes from scratch everytime you make a new website rather than a "drag and drop" build approach.

However, that doesn't mean I'm going to let the theme just gather dust. If developers request certain features that would assist them then I will add them accordingly.

Feel free to criticize my work and let me know how I can improve this WordPress theme by leaving feedback, that would be much appreciated because there's always room for improvement.

Obviously a big thanks to Jeremy Thomas, without this great framework this theme would not be possible.

-Tom Hartley
