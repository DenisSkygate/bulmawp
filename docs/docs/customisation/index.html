Customisation

As a development foundation, BulmaWP comes with only a few features built into the core theme. Now that you've made a child theme, it's easy to change any of the functionality that BulmaWP comes with to suit your own needs.

The Bulma framework is called through theÂ cdnjs CDN, this is an effort to make your WordPress website lightweight and fast. We've also included Font Awesome into the theme, just for your convenience.

As Bulma and Font Awesome get updated in the future, these will be automatically added to BulmaWP so that you can never worry about missing an update again.

Breadcrumbs

Breadcrumbs are built into BulmaWP using the markup that is provided within theÂ Bulma docs. On the BulmaWP theme settings page within your WordPress installation, you are able to modify the different breadcrumb features such as size, alignment and to include Font Awesome icons etc.

If you decide to use Font Awesome icons, there will be default ones already set for the homepage, category, tag and search result pages. For posts and pages, you can set these through the custom post meta box provided.

But if you simply don't want to use breadcrumbs on your site, you can easily disable them on the "BulmaWP Settings" page.

Footer

The footer is very simple to use, you can choose to use the pre-wrote copyright statement if you wish or simply write whatever you want to display in your footer.

Navbar

The navbar is straightforward and easy to change the styling of, Bulma offers a lot of customisation options which have been integrated well into the theme. You can add simple modifiers such asÂ is-fixed-top,Â is-fixed-bottom,Â is-transparentÂ andÂ is-spaced.

Page Templates

There is an extra page template which has been included, a full-width page which simply doesn't show the sidebar on the right side of the page.

Pagination

The pagination within BulmaWP cannot be disabled as it is a WordPress requirement, however, the pagination built into the theme is only enabled for viewing posts and not for comments. You are able to change the size, style, alignment, and the number of links shown easily without rebuilding the function yourself.

Search Bar

The default WordPress search bar markup has been changed to work with the Bulma framework, this has been placed in the searchform.php file within the directory of the BulmaWP theme. You can easily change the placeholder text, color of the search button and if you want to use an icon instead of text within the "BulmaWP Settings" page.

Social

The last customisable feature is adding social network icons to your navbar. You can add Facebook, Google Plus, Instagram, Pinterest, RSS and Twitter icons to your navbar which are shown as a child ofÂ navbar-end. If you do not wish any social network icons to show, just leave the fields blank on the "BulmaWP Settings" page.
